# VexaChain Consulting — Website

A complete single-page website for a supply chain and operations consultancy startup.

## Files

- `index.html` — Main HTML structure
- `style.css` — All styles and responsive design
- `script.js` — Chart.js chart, animations, and interactions

## Sections

1. **Navigation** — Sticky header with client login CTA
2. **Hero** — Split layout with live client dashboard mockup
3. **Stats Band** — Key proof points (engagements, savings, retention)
4. **Services** — Six practice areas in a grid layout
5. **Diagnostics** — Before/after operational health panels
6. **Case Studies** — Three detailed client outcomes
7. **Data Platform** — 4-step framework + live performance chart
8. **CTA** — Diagnostic call booking form
9. **Footer** — Links and copyright

## How to Use

Simply open `index.html` in any modern web browser. No build step or server required.

The chart requires an internet connection to load Chart.js from CDN. Alternatively, download Chart.js and reference it locally.

## Dependencies

- [Google Fonts — Syne & DM Sans](https://fonts.google.com)
- [Chart.js 4.4.1](https://www.chartjs.org)

## Customisation

- Colors: Edit CSS variables in `:root` inside `style.css`
- Content: All text and data is in `index.html`
- Chart data: Edit the `actual` and `target` arrays in `script.js`
